Corporate Identity

Corporate Identity หมายถึง อัตลักษณ์ขององค์กร ซึ่งเป็นสิ่งที่ออกแบบมาเพื่อให้สอดคล้องกับกลุ่มเป้าหมายทางธุรกิจ เป็นภาพลักษณ์ ที่สื่อถึงผู้บริโภค

ในการออกแบบอัตลักษณ์ มีองค์ประกอบสำคัญ 3 ประการคือ

ตราสัญลักษณ์องค์กร (Corporate Logo)
รูปแบบตัวอักษร (Font)
กำหนด ชื่อการค้า (Trade Name) ให้ใช้ Font DB Sathorn X
สำหรับ สินค้าของบริษัทฯ จะต้องใช้ Font นี้เท่านั้น ตัวอย่างเช่น
ราเชอร์ | โทมัฮอค | ฟรีดัคท์ 10%
สี (Color)
กำหนดให้ "สีน้ำเงิน" เป็นสีหลักในการทำสื่อ
Color (CMYK): C 100, M 70, Y 0, K 0
Color (RGB): R 0, G 79, B 159
Pantone Solid Color: Pantone 286 C
